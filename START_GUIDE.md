# 🚀 快速启动指南

## ✅ 前端已成功启动！

前端服务正在运行：http://localhost:5173

## 📋 启动步骤

### 1. 启动后端（如果还没启动）
```bash
cd src
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
```

### 2. 前端（已启动）
前端已在 http://localhost:5173 运行

## 🎯 功能测试

### 1. 访问系统
打开浏览器访问：http://localhost:5173

### 2. 测试批量上传功能
1. 点击左侧菜单 "数据管理"
2. 点击 "批量上传" 按钮
3. 下载示例文件：http://localhost:5173/sample_data.csv
4. 将文件拖拽到上传区域
5. 查看数据预览
6. 点击 "确认上传"

### 3. 其他功能
- **新增记录**：点击 "新增记录" 添加单条数据
- **批量删除**：勾选多条记录后批量删除
- **数据导出**：导出JSON格式数据
- **搜索筛选**：按监测点搜索数据
