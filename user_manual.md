## 1. 系统简介

### 1.1 系统概述
河流水质监控管理系统是一个专业的水质数据管理平台，用于监测、记录和分析河流水质指标。系统提供直观的数据可视化、智能报警和便捷的数据管理功能。

### 1.2 主要功能
- **仪表盘**：系统概览、KPI指标、趋势分析
- **数据管理**：水质记录的增删改查、批量操作
- **高级查询**：灵活的多条件查询、数据导出
- **报警监控**：超标报警、统计分析、趋势预警

### 1.3 监测指标
| 指标名称 | 单位 | 正常范围 | 说明 |
|----------|------|----------|------|
| 余氯 | mg/L | 0.5 - 4.0 | 消毒剂残留量 |
| 电导率 | µS/cm | ≤ 1000 | 水中导电能力 |
| pH值 | - | 6.5 - 8.5 | 酸碱度 |
| 氧化还原电位 | mV | ≥ 400 | 氧化还原能力 |
| 浊度 | NTU | ≤ 5.0 | 水体浑浊程度 |

## 3. 功能详解

### 3.1 仪表盘

#### 3.1.1 KPI指标卡片
仪表盘顶部显示四个关键指标卡片：
- **总记录数**：系统中的水质记录总数
- **平均pH值**：所有记录的平均pH值
- **平均浊度**：所有记录的平均浊度值
- **报警数量**：当前超标报警总数

每个卡片显示：
- 当前数值（带动画效果）
- 较昨日变化趋势
- 趋势图标和百分比

#### 3.1.2 趋势图表
**水质指标趋势图**：
- 显示最近7天的水质变化趋势
- 支持多指标同时显示
- 可切换显示全部或单个指标
- 鼠标悬停查看详细数值

**当前水质仪表盘**：
- 实时显示最新监测数据
- 五个仪表盘分别对应不同指标
- 颜色编码显示状态（绿色=正常，黄色=警告，红色=超标）

#### 3.1.3 报警统计
**报警类型分布**：
- 饼图展示各类报警占比
- 直观显示主要问题类型
- 支持图表类型切换

**实时报警列表**：
- 显示最新5条报警记录
- 包含监测点、时间、报警级别、超标项目
- 点击可查看详情

### 3.2 数据管理

#### 3.2.1 数据表格
表格显示所有水质记录，包含以下列：
- **选择框**：用于批量操作
- **ID**：记录唯一标识
- **监测点**：监测点标识（标签显示）
- **日期/时间**：监测时间
- **各项指标**：余氯、电导率、pH值、氧化还原电位、浊度
- **状态**：正常/警告/超标（标签显示）
- **操作**：编辑、删除按钮

#### 3.2.2 数据操作
**新增记录**：
1. 点击"新增记录"按钮
2. 填写表单信息：
   - 监测点ID（必填）
   - 日期和时间（必填）
   - 各项指标数值（必填）
3. 点击"确定"保存

**编辑记录**：
1. 点击记录行的"编辑"按钮
2. 修改需要更新的字段
3. 点击"确定"保存更改

**删除记录**：
1. 点击记录行的"删除"按钮
2. 确认删除操作
3. 系统自动删除记录

**批量删除**：
1. 勾选需要删除的记录
2. 点击"批量删除"按钮
3. 确认批量删除操作

#### 3.2.3 数据搜索和筛选
- **搜索**：在搜索框输入监测点关键词
- **排序**：点击表头进行升序/降序排序
- **分页**：使用底部分页控件浏览数据

#### 3.2.4 数据导出
1. 点击"导出数据"按钮
2. 系统自动生成JSON格式文件
3. 文件包含完整数据和时间戳

### 3.3 高级查询

#### 3.3.1 查询条件构建
系统支持动态构建查询条件：

**添加条件**：
1. 点击"添加条件"按钮
2. 选择字段类型（监测点、日期、各项指标）
3. 选择操作符（等于、大于、小于、介于等）
4. 输入查询值

**操作符说明**：
- **等于**：精确匹配
- **不等于**：排除指定值
- **大于/大于等于**：数值范围查询
- **小于/小于等于**：数值范围查询
- **包含**：文本模糊匹配
- **介于**：数值区间查询

**条件组合**：
- 支持多个条件同时生效
- 所有条件为"AND"关系
- 可删除不需要的条件

#### 3.3.2 查询执行
1. 设置好查询条件
2. 点击"查询"按钮执行
3. 系统显示符合条件的记录
4. 支持分页浏览结果

#### 3.3.3 结果操作
- **查看详情**：点击记录查看完整信息
- **编辑记录**：跳转到数据管理页面编辑
- **导出结果**：导出查询结果为JSON文件
- **刷新**：重新执行查询

#### 3.3.4 查询示例
**示例1：查询某监测点一周数据**
- 字段：监测点，操作符：等于，值：监测点001
- 字段：日期，操作符：介于，最小值：2024-03-01，最大值：2024-03-07

**示例2：查询pH值异常数据**
- 字段：pH值，操作符：小于，值：6.5
- 或
- 字段：pH值，操作符：大于，值：8.5

### 3.4 报警监控

#### 3.4.1 报警统计
**统计卡片**：
- **总报警数**：系统历史报警总数
- **警告级别**：黄色警告数量
- **严重级别**：红色严重报警数量
- **今日新增**：当天新增报警数

#### 3.4.2 报警分析
**报警类型分布**：
- 饼图/柱状图展示
- 显示各项指标超标次数
- 帮助识别主要问题

**报警趋势**：
- 最近7天报警数量变化
- 预测报警发展趋势
- 支持决策制定

#### 3.4.3 报警列表
**列表显示**：
- 监测点位置
- 报警时间
- 报警级别（警告/严重）
- 超标项目列表
- 主要指标数值

**筛选功能**：
- 按报警级别筛选（全部/警告/严重）
- 支持分页浏览
- 实时刷新数据

#### 3.4.4 报警详情
点击报警记录查看详细信息：
- **基本信息**：监测点、时间、级别
- **完整数据**：所有指标的详细数值
- **超标标识**：明显标注超标项目
- **阈值标准**：查看正常范围标准

## 4. 操作指南

### 4.1 日常工作流程

#### 4.1.1 数据录入流程
1. **登录系统** → 进入"数据管理"页面
2. **点击"新增记录"** → 打开录入表单
3. **填写监测信息**：
   - 输入监测点ID
   - 选择监测日期和时间
   - 输入各项指标数值
4. **数据验证** → 系统自动检查数据合理性
5. **保存记录** → 完成数据录入

#### 4.1.2 数据检查流程
1. **查看仪表盘** → 了解系统整体状态
2. **检查报警列表** → 查看是否有新的超标报警
3. **分析趋势图** → 观察水质变化趋势
4. **处理异常数据** → 对异常情况进行标记或修正

#### 4.1.3 报警处理流程
1. **发现报警** → 通过仪表盘或报警页面
2. **查看详情** → 点击报警记录查看详细信息
3. **分析原因** → 结合历史数据分析原因
4. **采取措施** → 现场检查或调整监测参数
5. **记录处理** → 在系统中标记处理结果

### 4.2 数据管理最佳实践

#### 4.2.1 数据录入规范
- **监测点命名**：使用统一的命名规范（如：监测点001）
- **时间记录**：准确记录监测时间，避免时间错误
- **数值精度**：按照仪器精度记录，不要过度四舍五入
- **及时录入**：监测完成后尽快录入，避免数据积压

#### 4.2.2 数据质量控制
- **范围检查**：录入时注意数值是否在合理范围内
- **趋势检查**：与历史数据对比，发现异常及时核实
- **完整性检查**：确保所有必填字段都有数据
- **一致性检查**：同一监测点的数据应保持一致性

#### 4.2.3 数据备份建议
- **定期备份**：建议每周备份一次数据
- **导出备份**：使用导出功能保存数据副本
- **版本管理**：保留多个时间点的数据版本
- **异地存储**：重要数据应在不同地点存储

### 4.3 系统维护

#### 4.3.1 日常维护
- **清理缓存**：定期清理浏览器缓存
- **检查更新**：关注系统版本更新
- **性能监控**：观察系统响应速度
- **日志检查**：如有权限，检查系统运行日志

#### 4.3.2 故障处理
- **页面无响应**：刷新页面或重新登录
- **数据加载失败**：检查网络连接
- **操作失败**：确认操作是否符合规范
- **系统异常**：联系系统管理员

## 5. 常见问题

### 5.1 数据录入问题

**Q: 为什么无法保存记录？**
A: 请检查以下项目：
- 所有必填字段是否都已填写
- 数值是否在合理范围内
- 网络连接是否正常

**Q: 录入的数据显示为红色？**
A: 红色表示该数值超出正常范围，属于超标数据。请核实监测数据是否准确。

**Q: 如何修改已录入的数据？**
A: 在数据管理页面找到对应记录，点击"编辑"按钮进行修改。

### 5.2 查询功能问题

**Q: 查询没有结果怎么办？**
A: 检查查询条件是否合理：
- 条件是否过于严格
- 日期范围是否正确
- 数值范围是否合理

**Q: 如何查询特定时间段的数据？**
A: 使用"介于"操作符，设置开始和结束日期。

**Q: 可以保存查询条件吗？**
A: 当前版本不支持保存查询条件，建议截图保存重要查询设置。

### 5.3 报警功能问题

**Q: 什么情况下会触发报警？**
A: 当水质指标超出预设阈值时会触发报警：
- 余氯 < 0.5 或 > 4.0 mg/L
- 电导率 > 1000 µS/cm
- pH值 < 6.5 或 > 8.5
- 氧化还原电位 < 400 mV
- 浊度 > 5.0 NTU

**Q: 警告和严重有什么区别？**
A: 
- **警告**：数值略超出正常范围
- **严重**：数值大幅超出正常范围（超过阈值1.5倍）

**Q: 如何消除报警？**
A: 报警是基于数据自动生成的，需要通过录入新的正常数据来改善报警状态。

### 5.4 系统使用问题

**Q: 忘记密码怎么办？**
A: 联系系统管理员重置密码。

**Q: 系统运行缓慢怎么办？**
A: 尝试以下方法：
- 清理浏览器缓存
- 减少同时打开的页面
- 检查网络连接速度

**Q: 数据导出失败怎么办？**
A: 检查以下项目：
- 数据量是否过大
- 浏览器是否阻止下载
- 磁盘空间是否充足

### 5.5 移动端使用

**Q: 手机上能正常使用吗？**
A: 系统支持响应式设计，可在手机浏览器上正常使用，但部分复杂操作建议使用电脑。

**Q: 移动端功能有什么限制？**
A: 移动端功能完整，但表格显示和复杂查询操作体验可能不如电脑端。

## 7. 数据备份

### 7.1 手动备份
1. 进入"数据管理"页面
2. 点击"导出数据"按钮
3. 保存导出的JSON文件
4. 建议文件名包含日期：`water_quality_2024_03_11.json`

### 7.2 备份文件说明
导出的JSON文件包含：
- **metadata**：导出时间、记录总数等信息
- **data**：完整的水质记录数据
- **schema**：数据结构说明

### 7.3 数据恢复
如需恢复数据，请联系系统管理员，提供备份文件进行数据导入。

### 7.4 备份频率建议
- **日常备份**：重要数据建议每日备份
- **周备份**：每周进行一次完整备份
- **月备份**：每月进行一次归档备份
